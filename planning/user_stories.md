# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

[👉🏾👉🏾👉🏾 Include at least at least 1, but no more than 3, user roles.]
1. Gardener: someone who has a garden and looking for flowers to grow
2. Flower enthusiast/florist: Somone interested in flowers, floriography, and taking pictures of flowers
3. Learner: Someone wanting to learn about flowers or what types to get for an occasion

## Draft User Stories

[👉🏾👉🏾👉🏾 Include at least at least 10 user stories in this format:]

1. As a [Gardener], I want to [search for flowers by sunlight and soil needs], so that [I cna choose plants that will thrive in my garden].
2. As a [Florist], I want to [view symbolic meaning of each flower], so that [I can create bouquets that convey the right message ].
3. As a [Flower Enthusiast], I want to [create an account so i can browse flower by catagories], so that [I can find species that match my astetic preferences]. 
4. As a [Beginner Gardener], I want to [step by step growing guides], so that [I cna learn how to care for flowers without feeling overwhelmed].
5. As a [Gardner], I want to [create an account and save my favorite flowers], so that [I can easily revisit the ones I love/ plan to grow].
6. As a [Learner], I want to [explore flowers by meaning], so that [that I can choose the perfect flower for an occasion]. 
7. As a [Gardener], I want to [know the toxicity level of each flower], so that [I can keep my family safe].
8. As a [Florist], I want to [upload photos of bouquets I create], so that [I can recreate them quickly/ show customers].
9. As a [Flower Enthusiast], I want to [see where flowers naturally grow], so that [I can identify species when I encounter them]. 
10. As a [Learner], I want to [have a clean and intuitive interface], so that [I cna quickly find the information I need whithout confusion].
